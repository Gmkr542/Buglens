# BugLens — AI Code Debugger

A FastAPI application that analyzes code with Groq LLM and includes a simple chatbot.

## Architecture

```
app.py                  ← FastAPI entry point (mounts all routers)
routes/
  debug.py              ← POST /debug/        — AI code analysis
                           GET  /debug/status  — Groq connectivity check
  chatbot.py            ← POST /chatbot/       — Send message, get reply
                           POST /chatbot/clear  — Clear chat history
                           GET  /chatbot/history— Fetch history
services/
  groq_service.py       ← Groq API (llama-3.3-70b-versatile)
  gemini_service.py     ← Gemini API (optional)
  llama_service.py      ← Thin wrapper over groq_service
utils/
  parser.py             ← Robust JSON extractor (brace-counting)
templates/
  index.html            ← Main debugger UI
  chat.html             ← Chatbot UI
```

## Quick Start

**1. Install dependencies:**
```bash
pip install -r requirements.txt
```

**2. Set environment variables:**
```bash
# Groq (required) — https://console.groq.com/keys
export GROQ_TOKEN="your_groq_token"

# Optional overrides
export GROQ_URL="https://api.groq.com/openai/v1/chat/completions"
export GROQ_MODEL="llama-3.3-70b-versatile"
export GROQ_VERIFY_SSL="true"

# Gemini (optional) — https://aistudio.google.com/app/apikey
export GEMINI_API_KEY="your_gemini_key"
```

**3. Run the app:**
```bash
python app.py
```
Or with uvicorn directly:
```bash
uvicorn app:app --host 127.0.0.1 --port 5000 --reload
```

**4. Open in browser:**
- Debugger: http://127.0.0.1:5000
- Chatbot:  http://127.0.0.1:5000/chat
- API docs: http://127.0.0.1:5000/docs   ← auto-generated by FastAPI

## API Reference

### Debug Endpoint
```bash
# Analyze code
curl -X POST http://127.0.0.1:5000/debug/ \
  -H "Content-Type: application/json" \
  -d '{"code": "print(1+1)", "apiKey": "YOUR_GROQ_TOKEN"}'

# Check connectivity
curl http://127.0.0.1:5000/debug/status
```

Response schema:
```json
{
  "issues":      [{"line": 5, "severity": "error", "type": "SyntaxError", "message": "..."}],
  "has_errors":  true,
  "analysis":    "Overall code quality summary...",
  "fixed_code":  "corrected source here",
  "improvements": ["Use type hints", "Add input validation"],
  "score":       72,
  "summary":     {"errors": 1, "warnings": 2, "lines": 30},
  "provider":    "groq"
}
```

### Chatbot Endpoints
```bash
# Send message
curl -X POST http://127.0.0.1:5000/chatbot/ \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!"}'

# Clear history
curl -X POST http://127.0.0.1:5000/chatbot/clear

# Get history
curl http://127.0.0.1:5000/chatbot/history
```

## Logs

- `logs/buglens.log`      — Human-readable rotating log (max 5 MB × 3 backups)
- `logs/buglens.json.log` — Structured JSON log for machine parsing

## Troubleshooting

- **401 Unauthorized** — Groq token invalid; regenerate at https://console.groq.com/keys
- **SSL errors** — Set `GROQ_VERIFY_SSL=false` temporarily for debugging
- **DNS errors** — Check internet access to `api.groq.com`
- **Proxy** — Set `HTTP_PROXY` / `HTTPS_PROXY` environment variables
